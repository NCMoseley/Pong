# Pong Game

A basic Javascript pong game using SVGs.

## Setup

**Install dependencies:**

`> npm i`

**Run locally with Webpack Dev Server:**

`> npm start`

**Build for production:**

`> npm run build`

## Keys

*SpaceBar: Pause
* m: Music
* f: Easter egg

**Player 1:**
* a: up
* z: down

**Player 2:**
* ▲ : up
* ▼: down
